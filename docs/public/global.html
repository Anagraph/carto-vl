RSys

An RSys defines a local coordinate system that maps the coordinates in the range -1 <= x <= +1; -1 <= y <= +1 to an arbitrary rectangle in an external coordinate system. (e.g. Dataframe coordinates to World coordinates) It is the combination of a translation and anisotropic scaling.
