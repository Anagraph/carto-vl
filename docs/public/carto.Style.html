Class: Style

carto.Style(definition)

new Style(definition)

A Style defines how the data will be displayed: the color of the elements and size are basic things that can be managed through styles. Styles also control the element visibility, ordering or aggregation level.

A Style is created from an styleSpec object or from a string. Each attribute in the styleSpec must be a valid expression. Those expressions will be evaluated dinamically for every element in the dataset.

Parameters:
Name Type Description
definition string | StyleSpec

The definition of a style. This parameter could be a string or a StyleSpec object
Fires:
  • event:CartoError
Examples

Create a style with black dots using the string constructor 

new carto.Style(`
  color: rgb(0,0,0)
`);

Create a style with black dots using the styleSpec constructor 

new carto.Style({
  color: carto.style.expression.rgb(0,0,0)
});
