(constant) RTT_WIDTH

The Render To Texture Width limits the maximum number of features per tile: maxFeatureCount = RTT_WIDTH^2

Large RTT_WIDTH values are unsupported by hardware. Limits vary on each machine. Support starts to drop from 2048, with a drastic reduction in support for more than 4096 pixels.

Large values imply a small overhead too.
