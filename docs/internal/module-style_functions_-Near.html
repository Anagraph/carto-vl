Class: Near

style/functions/~Near(input, center, threshold, falloff)

new Near(input, center, threshold, falloff)

Near returns zero for inputs that are far away from center. This can be useful for filtering out features by setting their size to zero.

Parameters:
Name Type Description
input *
center *
threshold *

size of the allowed distance between input and center that is filtered in (returning one)
falloff *

size of the distance to be used as a falloff to linearly interpolate between zero and one
